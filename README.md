# 离谱票 - 全网最低价一键比价跳转器 🎫

> 🌐 **在线使用：https://maoshaoxing.github.io/crazy-tickets/** ← 点这里直接打开！

不是爬虫，是**一键比价跳转器**——输入出发地、目的地、日期，一键打开所有主流票务平台的搜索结果页，对比实时价格后直接预订支付。

## ✨ 特点

- ✅ **完全合法** — 只生成 URL，不抓取任何平台数据
- ✅ **价格实时** — 直接跳转平台，看到的就是最新的真实价格
- ✅ **零后端零维护** — 纯静态 HTML，托管即用
- ✅ **含燃油费** — 机票跳转到平台页面，显示的就是含税含燃油最终价
- ✅ **扫码支付** — 从平台页面直接下单，微信/支付宝扫码支付

## 🚀 支持的平台

| 类型 | 平台 |
|------|------|
| 🚄 火车票 | 12306（官方）、携程、去哪儿、飞猪、美团、同程、拼多多 |
| ✈️ 机票 | 携程、去哪儿、飞猪、美团、同程、拼多多 |

> 注：12306 和携程支持 URL 参数直达搜索结果；其余平台可能需在跳转后点一次「搜索」或手动输入。

## 📦 使用方式

直接访问 **https://maoshaoxing.github.io/crazy-tickets/**

### 本地打开
```bash
# 双击或用浏览器打开
open index.html
```

## 📱 使用流程

1. 选择「高铁/火车票」或「机票」
2. 输入出发地、目的地
3. 选择出发日期（默认明天）
4. 点击「比价」
5. 点击任意平台卡片 → 在新标签页打开搜索结果
6. 或点「🚀 一键打开所有平台」 → 同时打开所有平台
7. 对比价格 → 选最低价 → 填乘客信息 → 扫码支付 ✅

- 本工具不抓取、不存储任何平台数据
- 所有价格以各平台实时显示为准
- 部分平台（去哪儿/飞猪/美团/同程/拼多多）跳转后可能需再点一次「搜索」
